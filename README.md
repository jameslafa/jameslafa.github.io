# James Lafa - Personal Website

Personal portfolio showcasing my career journey as a CTO & Engineering Leader based in Berlin.

**Live:** https://jameslafa.github.io

## About

Interactive timeline featuring my professional experience, skills, and language proficiencies. Built with a focus on clean design, smooth animations, and bilingual support.

## Features

- **Bilingual Support (EN/DE)**: Automatic language detection with manual override
- **Interactive Timeline**: Journey through work experience and life milestones
- **Responsive Design**: Mobile-first approach with hamburger navigation
- **Smooth Animations**: Wind farm scene with hover interactions
- **Print-Friendly**: Optimized layout for resume printing

## Tech Stack

- Vanilla JavaScript (no frameworks)
- CSS3 with custom animations
- Static site hosted on GitHub Pages
- No build process or dependencies

## Design

- Dark theme with blue accent colors (#6ba3d6)
- Google Fonts (Outfit)
- Custom SVG illustrations
- Minimal and professional aesthetic
